Vou adicionar campos específicos no modelo `Lead` para armazenar a análise da IA e atualizar a interface para exibi-los.

### 1. Alteração no Banco de Dados (Schema)

Adicionarei os seguintes campos na tabela `leads` (`prisma/schema.prisma`):

* `aiPotential` (String?): Para classificar o potencial (ex: "Alto", "Médio", "Baixo").

* `aiScore` (Int?): Uma pontuação numérica de 0 a 100 para ordenação.

* `aiSummary` (String?): Um resumo explicativo da análise feita pela IA.

* `aiAction` (String?): A próxima ação recomendada (ex: "Ligar agora", "Enviar cupom").

* `aiLastAnalyzedAt` (DateTime?): Data e hora da última análise.

* aiMessageSugest (String) mEnsagem sugerida baseado no historico

### 2. Atualização dos Tipos (TypeScript)

Atualizarei o arquivo `src/types/index.ts` para incluir esses novos campos nas interfaces `Lead` e `UpdateLeadInput`, garantindo que o restante da aplicação reconheça os novos dados.

### 3. Atualização da Interface (Frontend)

* **Lista de Leads (`lead-list.tsx`)**: Adicionarei colunas para mostrar o "Score/Potencial" e a "Ação Sugerida".

* **Visualização Detalhada**: Adicionarei um tooltip ou uma seção expandida para ver o "Resumo da Análise" completo, já que texto longo não cabe bem na tabela.

### 4. Integração

Esses campos ficarão disponíveis no banco de dados para que sua automação n8n possa preenchê-los diretamente (via query SQL ou API, dependendo de como você conecta).

**Campos para seu agente n8n preencher:**

* `ai_potential` (texto)

* `ai_score` (número inteiro)

* `ai_summary` (texto)
