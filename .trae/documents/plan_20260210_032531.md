Implementar a página de Notificações Externas com visões para o Cliente e para o Admin.

### 1. Visão do Cliente (Usuário do Tenant)
Esta visão permitirá que o usuário veja as notificações do seu próprio tenant.

*   **Página:** Criar `src/app/(dashboard)/notifications/page.tsx`
    *   Server Component que busca notificações via `getNotifications`.
    *   Exibe título "Notificações".
    *   Renderiza `NotificationList`.

*   **Componente de Lista:** Criar `src/components/notifications/notification-list.tsx`
    *   Tabela com colunas:
        *   Data (`criadoEm` - formatado)
        *   Tipo (`notificationType` - badge)
        *   Cliente (`numeroCliente`)
        *   Agente (`numeroAgent`)
        *   Mensagem (Truncada, clique para expandir ou modal)
        *   Ações (Visualizar Detalhes)
    *   Modal de Detalhes para ver `userData` (JSON formatado) e mensagem completa.

*   **Navegação:** Atualizar `src/components/layout/sidebar.tsx`
    *   Adicionar item "Notificações" (ícone `Bell`) ao menu `tenantItems`.

### 2. Visão do Admin (Plataforma)
Esta visão permitirá que o Admin veja as notificações de um tenant específico através da tela de edição de usuário.

*   **Server Action Admin:** Atualizar `src/actions/admin.ts`
    *   Adicionar função `getUserNotifications(userId: string)` que:
        *   Busca o usuário.
        *   Descriptografa `databaseUrl`.
        *   Conecta no banco do tenant.
        *   Retorna as notificações (reutilizando a lógica de `getNotifications`, mas com contexto explícito).

*   **Página de Edição de Usuário:** Atualizar `src/app/(dashboard)/admin/users/[id]/page.tsx`
    *   Transformar o layout atual em abas (Tabs do shadcn/ui).
    *   **Aba 1: Dados do Usuário** (o formulário atual `UserForm`).
    *   **Aba 2: Notificações** (lista de notificações do tenant desse usuário).
        *   Usar o mesmo componente `NotificationList`, mas passando os dados buscados via `getUserNotifications`.

### Passos de Execução:
1.  Criar componente `NotificationList`.
2.  Criar página `/notifications` (Cliente).
3.  Atualizar `Sidebar`.
4.  Implementar `getUserNotifications` em `admin.ts`.
5.  Atualizar `/admin/users/[id]/page.tsx` com Tabs e a lista de notificações.
